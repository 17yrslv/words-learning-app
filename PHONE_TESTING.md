## 🚀 Запуск приложения

### 1. Выберите устройство
В Android Studio вверху справа:
- Нажмите на выпадающий список
- Выберите ваш телефон

### 2. Нажмите Run
- Зеленая кнопка ▶
- Или `Shift + F10`

### 3. Дождитесь сборки
- 3-5 минут первый раз
- Внизу будет показан прогресс

## ⚠️ Возможные проблемы

### Телефон не появляется в списке
**Решение:**
1. Проверьте, что отладка по USB включена
2. Отключите и подключите кабель заново
3. На телефоне нажмите "Разрешить отладку"

### Ошибка при сборке
**Решение:**
1. Скопируйте текст ошибки
2. Отправьте мне
3. Я быстро исправлю

### Приложение не устанавливается
**Решение:**
1. На телефоне разрешите установку из неизвестных источников
2. Или в настройках разрешите установку для Android Studio

### Приложение вылетает
**Решение:**
1. Проверьте версию Android (должна быть 7.0+)
2. Отправьте мне логи из Android Studio
