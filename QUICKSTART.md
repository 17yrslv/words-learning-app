## 🎯 Как использовать приложение

### Первый запуск
При первом запуске приложение автоматически импортирует 1999 английских слов из CSV файла. Это займет несколько секунд.

### Главный экран
- Показывает статистику: всего слов, изучается, выучено, на повторении
- Кнопка "🎯 Начать обучение" - переход к настройке сессии
- Кнопка "➕ Добавить слово" - добавление нового слова
- Кнопка "📈 Статистика" - просмотр детальной статистики

### Настройка сессии
1. Выберите режим: EN→RU, RU→EN или смешанный
2. Установите количество слов (5-50)
3. Выберите тип слов: новые, на повторении или все
4. Нажмите "Начать"

### Процесс обучения
1. Показывается слово (английское или русское)
2. Введите перевод с клавиатуры
3. Нажмите "Проверить" или Enter
4. Увидите результат + все варианты перевода
5. Нажмите "Далее" для следующего слова
6. Шкала прогресса показывает прогресс сессии

### Система интервальных повторений
- **Уровень 0**: новое слово (повторить сегодня)
- **Уровень 1**: через 1 день
- **Уровень 2**: через 3 дня
- **Уровень 3**: через 7 дней
- **Уровень 4**: через 14 дней
- **Уровень 5**: через 30 дней (выучено)

## 🔧 Возможные проблемы

### Gradle sync failed
**Решение**: 
- Проверьте подключение к интернету
- File → Invalidate Caches → Invalidate and Restart
- Убедитесь, что установлен JDK 17+

### CSV файл не импортируется
**Решение**:
- Проверьте, что файл `app/src/main/res/raw/words.csv` существует
- Размер файла должен быть ~117 KB
- Удалите приложение и установите заново

### Приложение не запускается
**Решение**:
- Проверьте минимальную версию Android (7.0+)
- Очистите проект: Build → Clean Project
- Пересоберите: Build → Rebuild Project
