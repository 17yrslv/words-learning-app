## ⚠️ Если что-то не так

### Gradle sync failed
→ Проверьте интернет
→ File → Invalidate Caches → Restart

### Нет устройств
→ Подключите телефон через USB
→ ИЛИ Tools → Device Manager → Create Device

### Ошибки компиляции
→ Скопируйте текст ошибки
→ Сообщите мне

### Приложение не запускается
→ Проверьте, что устройство выбрано
→ Попробуйте Build → Clean Project
