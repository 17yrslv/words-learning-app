# English Words Learning App

Android приложение для изучения английских слов с системой интервальных повторений (Spaced Repetition).

## Возможности

- ✅ **Два режима обучения**: EN→RU и RU→EN
- ✅ **Ввод ответа с клавиатуры** с гибкой проверкой
- ✅ **Система интервальных повторений** (6 уровней: 0, 1, 3, 7, 14, 30 дней)
- ✅ **Случайный порядок слов** в каждой сессии
- ✅ **Настраиваемое количество слов** (5-50 слов за сессию)
- ✅ **Показ всех вариантов перевода** после ответа
- ✅ **Шкала прогресса** текущей сессии
- ✅ **Статистика обучения**
- ✅ **Добавление новых слов**
- ✅ **Темная тема**
- ✅ **1999 слов** предзагружены из CSV
- ✅ **Работает полностью офлайн**

## Технологии

- **Язык**: Kotlin
- **UI**: Jetpack Compose + Material 3
- **База данных**: Room (SQLite)
- **Архитектура**: MVVM
- **Навигация**: Navigation Compose
- **Минимальная версия Android**: 7.0 (API 24)

## Как собрать и запустить

- JDK 17 или новее
- Android SDK с API 24+

## Использование приложения

### Первый запуск

При первом запуске приложение автоматически импортирует 1999 английских слов из CSV файла в локальную базу данных. Это займет несколько секунд.

### Главный экран

- Показывает статистику: всего слов, изучается, выучено, на повторении
- Кнопки для начала обучения, добавления слов и просмотра статистики

### Настройка сессии

- **Режим обучения**: EN→RU, RU→EN или смешанный
- **Количество слов**: от 5 до 50 (слайдер)
- **Тип слов**: новые, на повторении или все

### Процесс обучения

1. Показывается слово (английское или русское)
2. Вводите перевод с клавиатуры
3. Нажимаете "Проверить" или Enter
4. Видите результат (правильно/неправильно) + все варианты перевода
5. Нажимаете "Далее" для следующего слова
6. Шкала прогресса показывает текущий прогресс сессии

### Система интервальных повторений

Слова проходят через 6 уровней запоминания:
- **Уровень 0**: новое слово (повторить сегодня)
- **Уровень 1**: через 1 день
- **Уровень 2**: через 3 дня
- **Уровень 3**: через 7 дней
- **Уровень 4**: через 14 дней
- **Уровень 5**: через 30 дней (выучено)

При правильном ответе уровень повышается, при неправильном - сбрасывается на 0.

## Формат CSV файла

CSV файл находится в `app/src/main/res/raw/words.csv`:

```csv
english,russian
apple,"яблоко, яблочко"
cat,"кот, кошка, котик"
dog,"собака, пес, песик"
```

- Первая колонка: английское слово
- Вторая колонка: русские переводы (через запятую)
- Если перевод содержит запятую, он заключается в кавычки

## Архитектура

### MVVM Pattern

- **Model**: Room entities, Repository
- **View**: Composable screens
- **ViewModel**: State management, business logic

### Clean Architecture

- **Data Layer**: Database, Repository, CSV importer
- **Domain Layer**: Use Cases, Models
- **UI Layer**: Screens, Components, Navigation

### Основные компоненты

- **CheckAnswerUseCase**: Гибкая проверка ответа (игнорирует регистр, пробелы, знаки)
- **UpdateWordProgressUseCase**: Обновление прогресса слова по алгоритму Spaced Repetition
- **GetSessionWordsUseCase**: Получение слов для сессии в случайном порядке
