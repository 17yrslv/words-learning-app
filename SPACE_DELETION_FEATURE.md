# Функция удаления пространств

## Дата добавления: 2026-04-20

## Как использовать:

1. Откройте список пространств (нажмите на название текущего пространства)
2. **Зажмите** (долгое нажатие) на название пространства, которое хотите удалить
3. Появится диалог подтверждения удаления
4. Нажмите "Удалить" для подтверждения или "Отмена" для отмены

## Важные особенности:

### ✅ Защита пространства по умолчанию
- Пространство "English" (id=1) **нельзя удалить**
- При долгом нажатии на него ничего не произойдет
- Это защищает основной словарь от случайного удаления

### ✅ Автоматическое переключение
- Если вы удаляете **текущее активное** пространство
- Приложение автоматически переключится на пространство "English"
- Вы не останетесь без активного пространства

### ✅ Каскадное удаление
- При удалении пространства удаляются **ВСЕ** слова из него
- Это необратимая операция
- Диалог предупреждает об этом красным текстом

## Пример использования:

```
1. Создали пространство "Italian"
2. Добавили 50 итальянских слов
3. Решили удалить это пространство
4. Зажали "Italian" в списке пространств
5. Подтвердили удаление
6. Пространство и все 50 слов удалены
7. Если "Italian" было активным - переключились на "English"
```

## Безопасность:

- ✅ Диалог подтверждения предотвращает случайное удаление
- ✅ Предупреждение о безвозвратном удалении данных
