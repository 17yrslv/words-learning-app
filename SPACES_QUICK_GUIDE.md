# 🚀 Быстрая шпаргалка: Система пространств

## Дата: 2026-04-20

---

## 📱 Три простых действия:

### 1️⃣ Создать пространство
```
Нажать на название → "+ Создать новое пространство" → Ввести название → Создать
```

### 2️⃣ Переключиться
```
Нажать на название → Выбрать из списка
```

### 3️⃣ Удалить
```
Нажать на название → ЗАЖАТЬ нужное пространство → Подтвердить
```

---

## ⚡ Горячие клавиши:

| Действие | Как сделать |
|----------|-------------|
| Открыть список | Нажать на название пространства (слева сверху) |
| Выбрать | Короткое нажатие |
| Удалить | Долгое нажатие (зажать) |
| Создать | Кнопка "+ Создать новое пространство" |

---

## 🛡️ Защита:

- ❌ "English" нельзя удалить
- ✅ Диалог подтверждения при удалении
- ✅ Автоматическое переключение при удалении текущего

---

## 💡 Примеры названий:

- **Языки:** Italian, Spanish, German, French, Japanese
- **Темы:** Business, Travel, Technical, Medical
- **Уровни:** Beginner, Intermediate, Advanced
- **Проекты:** Work, School, Hobby

---

## 🎯 Что помнить:

1. Каждое пространство = отдельный словарь
2. Слова не смешиваются между пространствами
3. Статистика считается отдельно
4. Удаление = безвозвратно
5. Обновление списка = мгновенно
