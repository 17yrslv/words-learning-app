## Как это работает:

1. **При первом запуске:**
   - Выполняется миграция базы данных
   - Создается пространство "English" с id=1
   - Все импортированные слова добавляются в это пространство

2. **Создание нового пространства:**
   - Пользователь нажимает на кнопку с названием текущего пространства
   - Выбирает "+ Создать новое пространство"
   - Вводит название (например, "Italian", "Spanish", "German")
   - Новое пространство создается и сразу появляется в списке
   - Приложение автоматически переключается на новое пространство

3. **Переключение между пространствами:**
   - Нажать на кнопку с названием пространства
   - Выбрать нужное пространство из списка
   - Все данные (слова, статистика) обновляются для выбранного пространства

4. **Добавление слов:**
   - Новые слова добавляются в текущее активное пространство
   - Слова из разных пространств не смешиваются

## Технические особенности:

- **Реактивность:** Используется Room Flow для автоматического обновления UI
- **Производительность:** Два независимых Flow для списка пространств и текущего пространства
- **Изоляция:** Каждое пространство полностью изолировано от других
- **Миграция:** Безопасная миграция существующих данных
