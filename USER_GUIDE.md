## 🚀 Запуск приложения

### В Android Studio:
1. Нажмите зеленую кнопку **▶ Run** (или `Shift + F10`)
2. Дождитесь сборки (первый раз 3-5 минут)
3. Приложение запустится на устройстве

### Первый запуск:
- Приложение автоматически импортирует **1999 английских слов** из CSV
- Это займет 5-10 секунд
- Откроется главный экран со статистикой

## 📱 Как пользоваться приложением

### Главный экран
- Показывает статистику: всего слов, изучается, выучено, на повторении
- **"🎯 Начать обучение"** - начать сессию
- **"➕ Добавить слово"** - добавить новое слово
- **"📈 Статистика"** - детальная статистика

### Настройка сессии
1. **Режим обучения:**
   - EN → RU (английский → русский)
   - RU → EN (русский → английский)
   - Смешанный режим

2. **Количество слов:** 5-50 (слайдер)

3. **Тип слов:**
   - Новые слова
   - На повторении
   - Все слова

4. Нажмите **"Начать"**

### Процесс обучения
1. Показывается слово (английское или русское)
2. Введите перевод с клавиатуры
3. Нажмите **"Проверить"** или `Enter`
4. Увидите результат:
   - ✅ **Правильно!** - показываются все варианты перевода
   - ❌ **Неправильно** - показывается ваш ответ и правильные варианты
5. Нажмите **"Далее"** для следующего слова
6. Шкала прогресса вверху показывает прогресс сессии (например, 8/20)

### Результаты сессии
- Показывает статистику: правильно/неправильно, процент точности
- Список слов с ошибками
- Кнопка **"Повторить ошибки"** - повторить только слова с ошибками
- Кнопка **"На главную"** - вернуться на главный экран

## 🧠 Система интервальных повторений

Слова проходят через **6 уровней запоминания**:

| Уровень | Интервал | Статус |
|---------|----------|--------|
| 0 | Сегодня | Новое слово |
| 1 | 1 день | Начальное запоминание |
| 2 | 3 дня | Краткосрочная память |
| 3 | 7 дней | Среднесрочная память |
| 4 | 14 дней | Долгосрочная память |
| 5 | 30 дней | **Выучено!** |

**Логика:**
- ✅ Правильный ответ → уровень +1
- ❌ Неправильный ответ → уровень сбрасывается на 0

## 🎨 Особенности приложения

### Гибкая проверка ответов
Приложение игнорирует:
- Регистр букв (Apple = apple)
- Лишние пробелы
- Знаки препинания (.,!?;:)

Принимает **любой вариант** из списка переводов.

### Случайный порядок
Слова в каждой сессии идут в случайном порядке для лучшего запоминания.

### Темная тема
Приложение использует темную тему Material 3 для комфортного использования.

## ❓ Частые вопросы

**Q: Как добавить свои слова?**
A: Нажмите "➕ Добавить слово" на главном экране, введите английское слово и русские переводы через запятую.

**Q: Как изменить слова в CSV?**
A: Отредактируйте файл `app/src/main/res/raw/words.csv`, удалите приложение и установите заново.

**Q: Можно ли экспортировать прогресс?**
A: В текущей версии нет, но можно добавить эту функцию.

**Q: Работает ли приложение без интернета?**
A: Да, приложение полностью офлайн.

**Q: Как сбросить прогресс?**
A: Удалите приложение и установите заново, или очистите данные в настройках Android.
