## 🧪 Тестирование функций

### При первом запуске:
1. **Импорт слов** (5-10 секунд)
   - Должен показаться экран "Загрузка слов..."
   - Затем откроется главный экран

2. **Главный экран**
   - Проверьте статистику: "Всего слов: 1999"
   - Должны быть кнопки: "Начать обучение", "Добавить слово", "Статистика"

### Тест 1: Начать обучение
1. Нажмите "🎯 Начать обучение"
2. Выберите режим: EN → RU
3. Установите количество слов: 10
4. Выберите "Все слова"
5. Нажмите "Начать"
6. **Проверьте:**
   - Показывается английское слово
   - Есть поле для ввода
   - Шкала прогресса вверху (1/10)

### Тест 2: Ввод ответа
1. Введите любой перевод (например, для "apple" введите "яблоко")
2. Нажмите "Проверить" или Enter
3. **Проверьте:**
   - Показывается результат (✅ или ❌)
   - Видны все варианты перевода
   - Есть кнопка "Далее"

### Тест 3: Завершение сессии
1. Пройдите все 10 слов
2. **Проверьте:**
   - Открывается экран результатов
   - Показана статистика (правильно/неправильно)
   - Есть список слов с ошибками
   - Кнопки "Повторить ошибки" и "На главную"

### Тест 4: Добавить слово
1. На главном экране нажмите "➕ Добавить слово"
2. Введите: English: "test", Russian: "тест"
3. Нажмите "Сохранить"
4. **Проверьте:**
   - Возврат на главный экран
   - Количество слов увеличилось на 1

### Тест 5: Статистика
1. Нажмите "📈 Статистика"
2. **Проверьте:**
   - Показана общая статистика
   - Есть прогресс-бар
   - Показан процент изучения

## ⚠️ Возможные проблемы

### Gradle sync failed
**Причина:** Нет интернета или проблемы с зависимостями
**Решение:** 
- Проверьте интернет
- File → Invalidate Caches → Restart

### Ошибка "SDK not found"
**Причина:** Android SDK не установлен
**Решение:**
- Tools → SDK Manager → Install Android SDK

### Приложение не запускается
**Причина:** Устройство не выбрано
**Решение:**
- Проверьте, что устройство выбрано в списке вверху
- Для эмулятора: запустите его сначала

### CSV не импортируется
**Причина:** Файл не найден
**Решение:**
- Проверьте: app/src/main/res/raw/words.csv существует
- Размер должен быть ~115 KB
